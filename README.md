# Название проекта: AutoPremium

---

## Назначение проекта

Разработать современный, адаптивный веб-сайт для автосалона **AutoPremium**, который позволяет:

- Просматривать каталог автомобилей с фотографиями, ценами и характеристиками  
- Фильтровать автомобили по марке, цене, году выпуска, типу топлива и коробке передач  
- Записываться на тест-драйв через форму с выбором даты и времени  
- Управлять каталогом автомобилей через административную панель (добавление, редактирование, удаление)  
- Просматривать и отменять заявки на тест-драйв (для администратора)
